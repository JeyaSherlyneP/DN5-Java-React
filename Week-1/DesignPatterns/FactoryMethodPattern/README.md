# Factory Method Pattern

## Objective

Implement the Factory Method Design Pattern in Java.

## Description

The Factory Method Pattern creates objects without exposing the object creation logic to the client.

## Files

- Notification.java
- EmailNotification.java
- SMSNotification.java
- NotificationFactory.java
- Main.java

## Supported Notification Types

- EMAIL
- SMS

javac *.java
