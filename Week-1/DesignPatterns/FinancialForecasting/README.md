# Exercise 7: Financial Forecasting

## Objective

Predict future financial values using recursion.

## Formula

Future Value = Present Value × (1 + Growth Rate)^Years

## Files

- ForecastCalculator.java
- Main.java

## Concepts Used

- Recursion
- Base Case
- User Input
- Financial Forecasting

## Example

Current Value = 100000

Growth Rate = 10%

Years = 5

Future Value = 161051.00

## How to Compile

```bash
javac *.java
```
