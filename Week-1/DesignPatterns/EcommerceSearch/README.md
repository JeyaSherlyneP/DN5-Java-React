# Exercise 2: E-commerce Platform Search Function

## Objective

Implement Linear Search and Binary Search to find products in an e-commerce catalog.

## Files

- Product.java
- SearchService.java
- Main.java

## Algorithms Used

1. Linear Search
2. Binary Search

## Time Complexity

| Algorithm      | Best Case | Average Case | Worst Case |
|----------------|-----------|---------------|-------------|
| Linear Search  | O(1)      | O(n)          | O(n)        |
| Binary Search  | O(1)      | O(log n)      | O(log n)    |

## How to Compile

```bash
javac *.java
```
